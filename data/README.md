# Data

This project uses publicly available wage data from Statistics Canada.

## Data Sources
The analysis draws on wage datasets from the Labour Force Survey and related open data tables for the following years:
- 2012
- 2016
- 2020
- 2024

## Notes on Availability
Raw datasets are not included in this repository due to size and licensing considerations.

All data used in the analysis can be accessed directly from Statistics Canada:
- Table 14-10-0417-01
- Labour Force Survey wage microdata

## Reproducibility
Data cleaning, inflation adjustment, and occupational classification steps are fully documented in the analysis notebook located in the `outputs/` directory.
